# 프로젝트: 온라인 쇼핑몰

## 개발환경
- ##### 개발 툴: Visual Studio Code
- ##### 개발언어: HTML5, JavaScript, jQuery
- ##### 디자인 API: Bootstrap, Font-awesome, CSS
- ##### 깃허브 주소: https://github.com/qkeb700

## 프로젝트 요약
- ##### 수량과 사이즈, 컬러에 따라 다르게 주문
- ##### 모든 데이터를 JSON 을 사용해 빠르게 속도처리
- ##### Ajax 를 사용한 효율적인 데이터 처리
